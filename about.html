About This Project

This project is part of an internship assignment at Keshav Codesoft. It focuses on using Bootstrap 5 to build clean and responsive web pages by combining UI components like navbars, cards, footers, and more.

I followed a structured approach to select layout patterns, build page compositions, and ensure responsiveness across devices.

This task helped me strengthen my fundamentals in Bootstrap and responsive design. I'm grateful to Keshav Codesoft for this opportunity to practice and build confidence in UI development.
